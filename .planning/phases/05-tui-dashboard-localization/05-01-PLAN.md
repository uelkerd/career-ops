4. Implement helper method `func (c *Catalog) FormatTimeAgo(dateStr string) string`:
   - Parse `dateStr` as `"2006-01-02"`. If invalid, return `dateStr`.
   - If difference is <= 0 days (today or future clamp), return `c.TimeToday`.
   - If difference is exactly 1 day, return `c.TimeYesterday`.
   - Otherwise return `fmt.Sprintf(c.TimeDaysAgo, days)`.

   - `func ToggleLang()`: If `Current == &En`, set `Current = &Tr`, otherwise `Current = &En`.
   - `func GetLang() string`: Return `"tr"` if `Current == &Tr`, else `"en"`.
7. Create `dashboard/internal/i18n/i18n_test.go` with unit tests covering `StatusLabel`, `FormatTimeAgo` for both `En` and `Tr`, and verifying `SetLang` / `ToggleLang` switch catalogs cleanly.
