# Tech in the City Database

This database serves as a reference for Career-Ops when evaluating roles, targeting specific companies, or generating customized applications in the Berlin/DACH region.

## 1. Job Profiles Database

When generating resumes or interview prep for these specific roles, emphasize these requirements and expectations:

### Salesforce Consultant
*   **Requirements:** No prior experience necessary (ideal for career changers), proficient computer skills, English language skills (at least B1), high school diploma or higher, strong communication skills.
*   **Tools / Topics:** Salesforce Cloud, CRM, Data management, Agile, Scrum.
*   **Expectations / Tasks:** Implement, customize, and optimize Salesforce solutions. Analyze client requirements, configure the platform, provide client training, and ensure continuous optimization of business processes.
*   **Salary Expectations:** €37,000 – €70,000 per year.

### AWS Cloud Engineer
*   **Requirements:** Beginner-friendly (ideal for career changers), no digital experience needed, basic computer/cloud knowledge, English (mind. B2), problem-solving, analytical thinking, team player.
*   **Tools / Topics:** AWS Core Services, Linux, Git, Python (OOP), Networking, Security, Databases, DevOps.
*   **Expectations / Tasks:** Design, implement, and manage robust AWS cloud infrastructures. Obtain AWS Cloud Practitioner Certification. Implement operational excellence in cloud environments.
*   **Salary Expectations:** €50,000 – €70,000 per year.

### Python Backend Developer
*   **Requirements:** No prior experience required, confident computer use, English (B1+), analytical thinking, ability to interpret data, organizational skills.
*   **Tools / Topics:** Python, Django, HTML, CSS, Git, Databases, APIs, Server-Client-Communication.
*   **Expectations / Tasks:** Develop and implement backend solutions for web applications, collaborate on projects using agile methods, manage databases and APIs.
*   **Salary Expectations:** €44,375 – €52,000 per year.

### Webentwickler:in (Web Developer)
*   **Requirements:** Ideal for career changers (no prior coding required), German or English (min. B1), confident computer use, creativity, self-organization.
*   **Tools / Topics:** HTML, CSS, JavaScript, various frameworks (React/Angular), Git, Databases.
*   **Expectations / Tasks:** Design, code, and test web pages/apps. Integrate server-client communications and manage databases. Consider performance optimization and security.
*   **Salary Expectations:** €45,000 – €65,000 per year.
